# NotFoundPage

Страница «Страница не найдена» (404).

- **Маршрут:** `*` (любой несуществующий путь)
- **Доступ:** публичный

## Описание

Fallback-страница для всех несопоставленных маршрутов. Сообщает пользователю, что запрошенная страница не существует.

## Состав

- `ui/NotFoundPage` — экран 404.

## Public API

```ts
export { NotFoundPage } from './ui/NotFoundPage';
```
