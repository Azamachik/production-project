# MainPage

Главная страница приложения.

- **Маршрут:** `/`
- **Доступ:** публичный

## Описание

Стартовая страница, на которую пользователь попадает при заходе в приложение. Демонстрационная страница с примерами общих UI-компонентов, счётчиком и списками (в проекте используется в том числе для отработки шаблонов). Загружается асинхронно (`.async.tsx`).

## Public API

```ts
export { MainPageAsync as MainPage } from './ui/MainPage.async';
```
