# ArticleEditPage

Страница создания и редактирования статьи.

- **Маршруты:** `/articles/create` (создание), `/articles/:id/edit` (редактирование)
- **Доступ:** только для авторизованных

## Описание

Единый компонент для двух сценариев: создание новой статьи и правка существующей (наличие `:id` в URL определяет режим). Заготовка под форму редактора статьи; обе route-точки ведут на один и тот же компонент.

## Состав

- `ui/ArticleEditPage` — страница редактирования/создания статьи.

## Public API

```ts
export { ArticleEditPage } from './ui/ArticleEditPage/ArticleEditPage';
```
