# Page

Виджет-обёртка контентной области страницы. Универсальный контейнер с бесконечным скроллом и сохранением позиции прокрутки.

## Описание

Оборачивает содержимое страниц в элемент `<main>` со скроллом. Реализует два сквозных сценария:

- **Бесконечная подгрузка** — через хук `useInfiniteScroll` и невидимый триггер-элемент вызывает `onScrollEnd`, когда пользователь долистывает до конца (используется на списке статей).
- **Сохранение скролла** — при монтировании восстанавливает позицию из фичи [ScrollSave](../../features/ScrollSave), а при прокрутке троттлингом (800 мс) сохраняет её по текущему пути.

Через `forwardRef` отдаёт DOM-элемент скролл-контейнера наружу (нужно, например, для `customScrollParent` у виртуализированного списка).

## Состав

- `ui/Page` — контейнер страницы с infinite scroll и восстановлением позиции.

## Public API

```ts
export { Page } from './ui/Page';
```
