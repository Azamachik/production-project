# PageLoader

Виджет полноэкранного индикатора загрузки. Показывается как fallback, пока грузится страница или чанк.

## Описание

Центрированный на весь экран спиннер (`Loader`). Используется в качестве `Suspense`-fallback при ленивой загрузке страниц в роутере, а также при асинхронной подгрузке модулей.

## Состав

- `ui/PageLoader` — контейнер, центрирующий лоадер на весь экран.

## Public API

```ts
export { PageLoader } from './ui/PageLoader';
```
