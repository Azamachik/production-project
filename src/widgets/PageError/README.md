# PageError

Виджет экрана ошибки. Fallback-заглушка, отображаемая при непойманном исключении в приложении.

## Описание

Показывается внутри `ErrorBoundary` из app-слоя, когда рендер падает с ошибкой. Сообщает пользователю о сбое и предлагает кнопку перезагрузки страницы (`location.reload()`).

## Состав

- `ui/PageError` — экран с сообщением об ошибке и кнопкой перезагрузки.

## Public API

```ts
export { PageError } from './ui/PageError';
```
