# Sidebar

Виджет бокового меню навигации. Сворачиваемая панель со ссылками на разделы и переключателями.

## Описание

Отображает список навигационных пунктов (`SidebarItem`), формируемых селектором `getSidebarItems` в зависимости от того, авторизован ли пользователь (некоторые ссылки, например профиль и статьи, доступны только авторизованным). Панель можно свернуть/развернуть. Внизу расположены фичи [ThemeSwitcher](../../features/ThemeSwitcher) и [LangSwitcher](../../features/LangSwitcher).

## Состав

- `ui/Sidebar` — контейнер бокового меню (кнопка сворачивания, переключатели темы и языка).
- `ui/SidebarItem` — отдельный пункт меню.
- `model/selectors/getSidebarItems` — список пунктов с учётом авторизации.
- `model/types/sidebar` — тип пункта меню.

## Public API

Экспорт вынесен в `ui/index.ts`:

```ts
export { Sidebar } from './Sidebar/Sidebar';
```
