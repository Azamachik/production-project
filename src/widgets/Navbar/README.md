# Navbar

Виджет верхней навигационной панели приложения. Отображается на всех страницах.

## Описание

Компонует несколько фич в единую шапку. Для неавторизованного пользователя показывает кнопку входа, открывающую [AuthByUsername](../../features/AuthByUsername). Для авторизованного — ссылку на создание статьи, колокольчик уведомлений ([ShowNotification](../../features/ShowNotification)) и меню пользователя ([AvatarDropdown](../../features/AvatarDropdown)). Данные о пользователе берутся из сущности [User](../../entities/User).

## Состав

- `ui/Navbar` — панель навигации, композиция фич авторизации, уведомлений и меню.

## Public API

```ts
export { Navbar } from './ui/Navbar';
```
