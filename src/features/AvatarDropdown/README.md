# AvatarDropdown

Фича выпадающего меню по аватару пользователя. Отображается в навбаре для авторизованного пользователя.

## Описание

При клике на аватар открывает `Dropdown` с навигационными пунктами: ссылка на профиль, ссылка в админку (только для ролей `ADMIN` / `MANAGER`) и выход из аккаунта (`logout`). Доступные пункты зависят от ролей пользователя, получаемых из сущности [User](../../entities/User).

## Состав

- `ui/AvatarDropdown` — меню действий пользователя.

## Public API

```ts
export { AvatarDropdown } from './ui/AvatarDropdown';
```
